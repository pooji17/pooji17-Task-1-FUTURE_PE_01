# Prompt 3 — CTA Sections (3 Variants)

## Purpose
Generates three distinct CTA sections: primary booking, urgency-based, and trust-building footer.

---

## Prompt

```
Write 3 call-to-action sections for [BUSINESS_NAME], a [BUSINESS_TYPE].

CTA 1: Primary booking/contact CTA (clear action, low commitment)
CTA 2: Urgency-based CTA (weekend, season, or limited availability angle)
CTA 3: Trust-building footer CTA (location, hours, welcoming tone)

Each CTA must include:
- Headline (max 10 words)
- 2–3 sentence supporting copy
- Button text (2–4 words)
- 2–3 trust signals (badges or bullet points)

Tone: [TONE]. Primary action: [BOOK / CALL / VISIT / ENQUIRE].
```

---

## Brewed Horizons Variables

| Variable | Value |
|---|---|
| BUSINESS_NAME | Brewed Horizons Cafe |
| BUSINESS_TYPE | specialty coffee café and brunch spot |
| TONE | friendly |
| PRIMARY ACTION | BOOK (table reservation) |
| LOCATION | 12, Road No. 36, Jubilee Hills, Hyderabad |
| HOURS | 8 AM – 6 PM daily |

## Additional CTA prompts (optional)

### Event enquiry CTA
```
Write a call-to-action section for private event enquiries at Brewed Horizons Cafe.
The café offers full buyouts for teams of 15+.
Include: headline, 2 sentences of supporting copy, a primary button ("Enquire about events"),
and 3 trust signals (response time, custom menus, minimum guests).
Tone: friendly but professional.
```

### Newsletter / loyalty CTA
```
Write a short CTA section inviting customers to join the Brewed Horizons loyalty
email list. Promise: early access to seasonal menu launches and a free coffee on
their first visit. Tone: warm. Length: 1 headline + 2 sentences + button text.
```
