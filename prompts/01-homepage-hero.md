# Prompt 1 — Homepage Hero Copy

## Purpose
Generates the headline, sub-headline, and intro paragraph for the homepage hero section.

---

## Prompt

```
You are a professional website copywriter specialising in local businesses.
Write homepage hero copy for [BUSINESS_NAME], a [BUSINESS_TYPE] located in [LOCATION].

Write:
- A headline (max 8 words) with a strong emotional hook
- A sub-headline (1–2 sentences) stating who it's for and the core benefit
- An intro paragraph (3–4 sentences) that builds trust and personality

Tone: [TONE: friendly / professional / bold]
Key differentiators: [LIST 2–3 THINGS THAT MAKE THEM UNIQUE]
Avoid: generic phrases like "world-class", "one-stop-shop", "passionate team".
```

---

## Variables

| Variable | Brewed Horizons Value |
|---|---|
| BUSINESS_NAME | Brewed Horizons Cafe |
| BUSINESS_TYPE | specialty coffee café and all-day brunch spot |
| LOCATION | Banjara Hills, Hyderabad |
| TONE | friendly |
| UNIQUE FACTORS | single-origin beans sourced directly from Coorg and Araku; all-day brunch menu that changes seasonally; work-friendly space with fast Wi-Fi and power at every seat |
