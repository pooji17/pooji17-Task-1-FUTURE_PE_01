# Prompt 4 — Tone Adaptation (Reusable for Any Client)

## Purpose
Takes existing copy generated by Prompts 1–3 and rewrites it for a different business type or tone register. Makes the entire framework reusable without rewriting prompts from scratch.

---

## Prompt

```
Rewrite the following website copy in a [NEW_TONE] tone, suitable for a [NEW_BUSINESS_TYPE].

Original copy:
[PASTE COPY HERE]

Tone guide:
- friendly (café / salon): warm, conversational, uses contractions, feels like a friend
- professional (clinic / agency): clear, confident, avoids slang, builds authority
- bold (gym / fast food): punchy, short sentences, uses second person, action-oriented

Keep: the core facts and structure.
Change: vocabulary, sentence length, and emotional register.
Output only the rewritten copy, no explanation.
```

---

## Example adaptations

### Café → Clinic (friendly → professional)
| | |
|---|---|
| Original | "Your mornings deserve something real." |
| Rewritten | "Reliable diagnostics. Clear results. On your schedule." |

### Café → Gym (friendly → bold)
| | |
|---|---|
| Original | "Come for the coffee. Stay because the vibe won't let you leave." |
| Rewritten | "Walk in. Push hard. Come back stronger." |

---

## Other business types this framework covers

| Business type | Tone | Swap these variables |
|---|---|---|
| Hair salon | friendly | services → haircut, balayage, treatment; location → your neighbourhood |
| Dental clinic | professional | products → check-up, cleaning, whitening; trust signals → certifications |
| Coaching institute | professional/bold | services → courses, mock tests, counselling; CTA → "Enrol now" |
| Restaurant | friendly | services → dine-in, takeaway, catering; hero hook → food-focused |
| Freelance agency | professional | services → branding, web design, social; CTA → "Start a project" |

---

## Tips for best results

1. Run Prompts 1–3 first to get the base copy for your chosen business
2. Use this prompt to create a second version in a different tone
3. Show the client both versions and let them choose — this doubles your perceived value
4. Save both versions in the client's folder for future updates
